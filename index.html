SSH Script

This was created when the original version of OSX Yosemite was released, which removed the "AlwaysAppendSearchDomains" functionality of the built-in DNS Resolver.

All this does, is an nslookup of the domain following the script, egreps out the IP address, and ssh's to that IP, for example:

./ssh.sh example.com

Will actually ssh, as your current user, to 93.184.216.34.
